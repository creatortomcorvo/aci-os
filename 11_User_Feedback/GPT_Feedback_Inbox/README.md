# GPT Feedback Inbox

Status: active manual inbox.

Purpose: temporary holding area for feedback from the live Chief Consigliere GPT.

Use one file per feedback item. Suggested filename:

`YYYY-MM-DD_short-pattern.md`

Example:

`2026-07-05_tc5-payment-chain-rendering.md`

Use the template:

`11_User_Feedback/GPT_Feedback_Record_Template_v0.1.md`

## Rules

- No real names.
- No client/counterparty names.
- No personal data.
- No privileged material.
- No live investigation facts.
- No confidential documents.
- No sanctions-screening reports with real parties.

## Review

Codex reviews inbox items and assigns:

- **ACT** - update ACI-OS now;
- **PARK** - useful later;
- **DISCARD** - not aligned or unsafe.

Nothing in this inbox becomes project memory until reviewed.
